Equipo

Daniel

Daniel se encargó de pensar y organizar cómo iba a ser la web. También se ocupó de su desarrollo, utilizando HTML, CSS y JavaScript para que todo funcionara y se viera bien. Además, configuró el repositorio en GitHub con varias ramas para que cada uno pudiera trabajar en su parte sin problemas y luego juntar todo.

Otra de sus tareas fue la implementación de la base de datos en el juego para poder cargar y guardar los datos.

Mónica

Mónica estuvo a cargo de la parte de programación del juego. Se encargó de crear y definir todas las funciones principales, incluida la función main y las 5 opciones que permitieron que el juego cobrara vida.

También tuvo que enfrentarse a problemas para conectar el programa con la base de datos, ya que PyCharm no actualizaba bien el entorno. Después de probar varios comandos en la terminal y descargar los paquetes necesarios, logró que todo funcionara correctamente.

Cristian

Cristian fue el encargado de crear y configurar la base de datos en Azure. Diseñó las tablas, definió sus relaciones para asegurar la consistencia de datos.

También preparó vistas específicas que permitían ver información clave como los resultados de las partidas, los jugadores y sus rankings. Todo esto ayudó a que el sistema fuera más completo y funcional.
